This plugin is the proxy component of the **Velocity Shop System**, running on the Velocity proxy server.

## Key Features
- **Message Relaying**: Relays price synchronization messages (`PRICE_UPDATE`) between multiple backend servers.
- **Channel Management**: Manages server communication via the `shopsystem:sync` channel.

## Installation
1. Place the `VelocityShopSystem-1.0.2.jar` file into the `plugins` folder of your Velocity server.
2. Restart the Velocity server.

## Configuration
No separate configuration file (`config.yml`) is required.
